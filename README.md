# qfin-projects

Algorithmic trading projects built for the UWA Quantitative Finance Society (QFin) competitions.

## Projects

### [sem1-trading-competition](./sem1-trading-competition/) - PanicTrader

1st place finish. Four strategies targeting distinct market inefficiencies across 8 simulated stocks - bid-ask spread regime detection, volatility spike exploitation, lead-lag confirmation trading, and multi-asset statistical arbitrage via linear regression. Python for research, C++ for parameter optimization (5,000x speedup).

### [sem2-trading-bot](./sem2-trading-bot/) - NIFTY Trader

Market-making bot for a turn-based simulated exchange. Built iteratively across 13+ strategy versions with ML-based direction prediction (Random Forest), momentum discovery, and adaptive position management. Includes a full research pipeline with Jupyter notebooks and automated model exports.
